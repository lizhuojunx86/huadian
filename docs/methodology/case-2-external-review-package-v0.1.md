## B. 审稿维度（核心）

### B.1 五个共通维度（三篇都要过）

| 维度 | 审什么 | 谁来审 |
|---|---|---|
| **制药 GMP / QA 实务** | 对"批生产记录 / 来料检验 / 反算行为"的刻画是否符合真实车间与质检实践 | 行业 GMP / QA 专家、内审 |
| **统计 / 方法论** | σ 检验、分布判定、子通道分层、代差量化的严谨性；**n=2 的推断边界** | 统计 / 计量学者 |
| **STS（科学技术与社会）** | "记录是博弈痕迹 / 合规叙事"这一核心立场的学理站位 | STS / 科学社会学学者 |
| **学术写作 / 投稿适配** | 论点清晰度、贡献陈述、目标渠道适配（作者写作偏弱，是已知短板）| 有发表经验的同行 |
| **引用规范** | 参考文献卷期页码（三篇已 web 最终化，仍请抽查）| 任一审稿人 |

### B.2 每篇专属维度 + 头号风险靶点

#### 09 — 分层合规叙事
- **STS / Latour ANT**：inscription / black-box / trial-of-strength 映射是否严谨；"AI 工程版 ANT"能否成立
- **组织社会学（Knorr Cetina / Van Maanen）**：三层 = 三套并存知识生产规范、realist vs confessional tale 的对应是否过度引申
- **知识管理（Wenger 实践共同体）**："深层须改组织结构、培训无法消除"的政策含义是否成立
- **治理（Scott legibility / métis）**：与 F-001 三层的对应是否精确；"多模态采集有理论上限"的论断
- **制药民族志**：深层"身体化经验"的**证据强度**（目前是访谈间接证据，深层 σ_real 为估计而非测量）
- 🎯 **头号风险**：三层划分的**可操作性**——如何客观判定某字段属于哪一层？（理论贡献越大，学理审查越严）

#### 08 — 双重反算
- **会计 / 盈余管理**：F-004"利润压线落入预期区间"、F-002"营收 upward"的跨域类比是否成立（Burgstahler & Dichev / Graham）
- **行为经济学 / 博弈论**：F-004 自保 framing（怕考核 + 怕调查）是否严谨；Prospect / agency 理论引用是否准确
- **制药 QA**：σ_floor 取自访谈（Q-027）的有效性；"σ≈0 + 紧贴边界 = 反算"判据在真实工艺下**是否会误伤优秀工艺**
- **STS / 伦理**："F-004 是一线人员理性行为而非道德问题"的 framing；F-002 跨文化普适性低于 F-004 的论断
- 🎯 **头号风险**：F-004 跨企业验证（X2）未做（已记为 documented unresolvable）——generalizability 是否靠"文献综述补强"足以说服外审

#### 10 — 子通道分层检测
- **信号检测理论 SDT（Green & Swets）**："可检测性 = 信噪比 / 审计 criterion 与真实 signal 错位"的映射是否严谨
- **统计（Simpson / 分层分析）**：§4 全方差分解的数学正确性；pooled vs within-subchannel；"何时必须分层"的阈值
- **法务会计（Benford-Nigrini）**："在量级无关的 Benford 传统上加分层维度"的定位是否准确、**有无前人已做**
- **信号理论（Spence / Zahavi）**："易操纵 ⊥ 易暴露"经 costly signaling 的解释（Zahavi handicap 有争议 / 以 Spence 为主是否够）
- **分析化学 / 制药 QC**：子通道边界值（main ≥30% / minor <6%）；定性化退化（"未过"/ 统一 0.1）与仪器 LOD 的关系
- 🎯 **头号风险**：子通道边界值暂定（n=2 / 押后 v0.5）——审稿可能要求边界的统计依据；反转悖论的前提（审计者同时看绝对 + 相对差）需严格界定适用条件

## C. 审稿次序（错峰 09 → 08 → 10）

```
2026-08 上旬   09 审稿窗（结构 / 地基）
2026-08 下旬   08 审稿窗（机制 / 建于 09 之上）
2026-09 上旬   10 审稿窗（检测 / 建于 08 之上）

2026-10        v1.0 三发布
```

**为何 09 先**：F-001（09）是 F-002/F-003/F-004 的共同结构性地基，先锁地基，下游 08/10 审稿不返工；09 理论权重最高（先啃硬骨头）；10 最新、最技术（数学可自证），放最后留沉淀窗。三篇审稿人池不同（09 偏 STS/社会学、08 偏经济/会计、10 偏统计/检测），错峰避免作者带宽被同时占满。

**备选**：若某篇外部审稿人档期先到，可局部调序——但 **09 不宜放最后**（地基后审会让 08/10 的前提悬空）。

## D. 请审稿人**权衡而非重新发现**的已知局限

以下三条作者已诚实自标，请评估"披露 + 补强是否充分"，而非当作新发现：

1. **n=2 推断边界**：全部结论建于**单企业、双产品**。跨产品稳健性仅证明"双产品内部一致"，跨企业为**有方向的假说**，非已证命题。
2. **识别 ≠ 证实**：方法论提供的是"结构筛查 / 第一道筛"——**提出嫌疑而非定罪**，不替代制度性确证。
3. **F-004 X2 跨企业访谈未做**：记为 documented unresolvable，靠文献综述补强。
