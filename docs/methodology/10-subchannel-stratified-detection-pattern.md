## 0. 论点与定位

**一句话论点**：在阈值制（≥ / ≤ / 区间）指标体系里，**操纵的可检测性是指标量级的函数**。统一阈值或汇总分析会系统性漏掉"最容易被操纵"的通道。检测必须按**量级同质的子通道**分层；不分层 = 把信号平均掉。

**本文的招牌贡献（signature contribution）**：「**易操纵 ⊥ 易暴露**」反转悖论 —— 越容易反算的通道（主成分型，紧贴 LOQ 看似合理），被操纵后信号越**弱**；越难反算的通道（微量型，微小绝对差却相对差极大），被操纵后信号越**强**。结果：朴素审计盯着"显眼"的微量通道（其实最可能真实），放过"不显眼"的主成分通道（其实最可能被操纵）——**检测注意力与操纵概率反向**。

这条悖论把 F-003 从"一个分组技巧"提升为一条**关于检测系统设计的一般原理**：任何按"异常显眼度"分配审计资源的系统，都会被量级结构系统性误导，除非先分层。

**三篇分工（避免与 08/09 重叠）**：
- **09 = 结构**（合规叙事是什么 / 三层代差）
- **08 = 机制 + 方向**（反算如何发生 / upward F-002 + downward F-004 / 整合识别树）
- **10 = 检测分辨率**（如何才"看得见"反算 / 为什么必须分层 / 量级如何决定可检测性）

三篇是同一案例的三个互补 lens：09 回答"记录是什么"，08 回答"反算往哪走"，10 回答"怎样才看得见"。

**四 finding 的 lens（案例护城河）**：
- **F-003**：本文主题，从 08 的工具升格为独立模式。
- **F-002**：F-003 是 F-002 上限 upward 通道的量级精细化；本文的分层对象正是 F-002 通道。
- **F-001**：分层检测是 F-001"二阶检验"（判定表层数据是否反算填表）的**分辨率工具**——F-001 说"表层可能是反算叙事"，本文给出"如何在表层里把反算看出来"的方法。

## 1. 引子 — 为什么需要量级分层

设想一个朴素的反算审计：把一份来料检验记录里所有"含量下限（≥）指标"收集起来，做一次统一的分布检验——算它们超过标准下限的"倍率"（实测 / 标准_min），看分布是否异常集中、是否过于贴线。直觉上这很合理：反算填表的人会把值写在"刚好过线"附近，所以贴线 + 低离散 = 嫌疑。

这个朴素做法会**系统性失败**，原因不在方向、而在分辨率。

在中药来料检验里，"含量下限指标"横跨极宽的量级：主成分指标的标准下限可达 ≥50%、≥86%（茯苓 β-葡聚糖 ≥50%、海螵蛸 CaCO₃ ≥86%、P2石膏 CaSO₄·2H₂O），微量有效成分的标准下限低到 ≥0.040%（延胡索乙素）。当你把它们混在一个桶里：
- 主成分指标倍率 **1.01–1.07x**（紧贴 LOQ）、跨 lot 几乎零波动；
- 微量指标倍率 **1.33–2.83x**（远超 LOQ）、跨 lot 天然离散。

把两者一起做分布检验，主成分的"贴线 + 零波动"这个最强反算信号，被微量成分的天然大离散**平均稀释**掉——整体看"分布健康、离散正常"，于是漏判。

**实证锚点**：lot-L1 单 lot，主成分子通道极差 ≈ 0.027，微量子通道极差 ≈ 9.98 → **极差比 ≈ 55:1**；跨 lot（L1→L2）扩大到 **136–370:1**。这个比值本身就是"汇总会抹掉多少信号"的直接度量。

**问题定性**：这是一个**检测分辨率**问题，不是检测方向问题（方向由 08 管）。同一种 upward 反算偏向，在不同量级通道呈现完全不同的可见度——所以检测的第一步不是"测",而是"先按量级把通道分开"。

## 2. F-003 命题精确化

**核心关系**：反算难度 ∝ 1 / |标准值 × 允许误差|（即"绝对 headroom"越小，越难无痕反算）。

反算者要做的，是写一个"看起来像真实测量、且刚好合规"的值。能不能做到，取决于标准量级给的**绝对操作空间**：

- **主成分型**（标准 ≥30%，实例 ≥50%/≥86%）：绝对 headroom 大。写 53.5%（标准 ≥50%）或 89.7%（标准 ≥86%），3–4 个百分点的"超出"看起来完全像一次正常测量的合理波动——**低调过线很容易**。
- **微量型**（标准 <6%，有效成分多在 <1%）：绝对 headroom 极小。标准 ≥0.040% 的指标，写 0.041% 会显得"可疑地精确贴线"，写 0.079%（实测）则相对差近一倍——**要么真测，要么大幅造假，没有"低调过线"的中间地带**。

**子通道划分（当前 / 边界暂定见 §10）**：

| 子通道 | 标准量级 | 实测倍率 | 反算难度 | 信号形态 |
|--------|---------|---------|---------|---------|
| 主成分型 main_component | ≥30%（实例 ≥50%/≥86%）| 1.01–1.07x | **低** | 紧贴 LOQ / 弱信号 |
| 浸出物型 extract | 2–30% | 1.25–1.68x | 中 | 中间形态 |
| 微量型 minor_component | <6%（有效成分）| 1.30–2.83x | **高** | 离散 / 强信号 |
| dual_limit（中间体/辅料/包材）| 区间制 | 紧贴中点 / 跨 lot 一致 | 低 | F-002 **不适用**（见 ADR-034）|

**与 08 的差异**：08 用这张表证明 F-002 反算随量级变化（工具性）；本文用它论证"量级决定可检测性"的一般原理，并发展出 §3 悖论、§4 数学、§5 协议——同一张表，08 是论据，10 是命题主体。

## 3. 检测悖论 — "易操纵 ⊥ 易暴露"（本文核心新贡献）

把 §2 的难度关系和 §1 的信号形态并置，浮现一个反直觉的结构：

**操纵难度与操纵后的暴露度反向。**

- 主成分型：**易操纵**（紧贴 LOQ 合理）→ 操纵后**难暴露**（信号弱，混在正常波动里）。
- 微量型：**难操纵**（微小绝对差难做得可信）→ 一旦操纵**易暴露**（信号强，要么贴得可疑、要么离谱）。

**推论（反直觉且有实操后果）**：朴素审计按"哪里看起来异常"分配注意力。但"看起来异常"的是微量通道（天然大离散、偶有极端值），而它其实最可能是真测；"看起来正常"的是主成分通道（贴线、平稳），而它其实最可能被反算。于是**审计注意力流向了最不需要的地方，放过了最该查的地方**——检测注意力与操纵概率系统性反向。

这就是为什么"分层"不是可选的优化，而是**必需**：不分层 = 用同一个"显眼度"门槛扫所有通道 = 把注意力交给信号强度 = 被反转悖论牵着走。分层 = 给每个量级通道一套**与该通道操纵难度匹配**的判据（主成分查"是否贴线且零波动"，微量查"是否异常紧贴或定性化退化"）。

**理论连接（signaling theory）**：经济学的信号理论（Spence 1973）和生物学的 handicap 原理（Zahavi 1975）共享一个内核——**难以伪造的信号才是可信信号**。把它倒过来用于审计：微量成分指标像一个"高伪造成本"的信号通道，正因为难以低调伪造，它的值更接近真实（伪造会暴露）；主成分指标像"低伪造成本"通道，正因为容易低调伪造，它的值最不可信。**可检测性是伪造成本的函数**——这给"易操纵 ⊥ 易暴露"反转提供了一个跨学科的理论根。（注：Zahavi handicap 原理在演化生物学内部有长期争议与修正；本文主要依托 Spence 的经济学信号模型，handicap 仅作生物学旁证。）

## 4. 信号稀释的数学 — 为什么"汇总"会抹掉信号

把反算检测形式化，就能看清"汇总"为何是错的。

**设定**：一个下限（≥）通道含若干指标，分布在子通道 \( s \in \{\text{main}, \text{extract}, \text{minor}\} \)。对每个指标定义反算诊断的两个量：倍率 \( r = \text{实测}/\text{标准}_{\min} \)，以及同物料同指标的跨 lot 离散 \( \sigma \)。

**反算的子通道指纹**：
- main：反算 → \( r \to 1.0^+ \)（贴线）且 \( \sigma \to 0 \)（跨 lot 几乎不动，因为"目标贴线值"固定）。
- minor：即便真测，\( r \) 也散布在 1.3–2.8、\( \sigma \) 天然大（真实药材含量受采收/储存波动）。

**汇总为何失败（全方差分解）**：若把整个通道当一个桶算总离散，按全方差定律

\[ \mathrm{Var}_{\text{pool}}(r) = \underbrace{\mathbb{E}_s[\mathrm{Var}(r \mid s)]}_{\text{组内平均方差}} + \underbrace{\mathrm{Var}_s(\mathbb{E}[r \mid s])}_{\text{组间均值离散}} \]

主成分反算的诊断信号是"\( \mathrm{Var}(r\mid \text{main}) \to 0 \) 且 \( \mathbb{E}[r\mid\text{main}] \to 1.0^+ \)"。但在 \( \mathrm{Var}_{\text{pool}} \) 里：
1. 第一项被 \( \mathrm{Var}(r\mid \text{minor}) \) 主导（微量天然大离散），主成分的"方差趋零"被淹没；
2. 第二项（组间均值差，main≈1.05 vs minor≈1.8）又把 pooled 方差进一步抬高。

结果：**pooled 统计量既不像 main、也不像 minor，主成分的贴线指纹无法从汇总数里还原**。这正是 Simpson 悖论一类的聚合失真——把异质分层塌缩成一张表，组内结构（main 紧、minor 松）被销毁，甚至可能反向（Simpson 1951）。

**极差比作为稀释度量**：\( \text{range}_{\text{minor}} / \text{range}_{\text{main}} = 55{:}1 \)（单 lot）到 \( 136\text{–}370{:}1 \)（跨 lot）。这个比值就是"汇总让微量通道的尺度压倒主成分信号"的倍率——比值越大，不分层的漏判越严重。

**判据（何时必须分层）**：当子通道间量级跨度导致 \( \mathrm{Var}_s(\mathbb{E}[r\mid s]) \) 或极差比超过某阈（case-2 实测 55–370），pooled 检验对最弱信号通道（main）的统计功效趋于丧失 → 必须分层。形式化阈值的标定留 v0.3 + 跨案例数据。

## 5. 整合 — 分层检测协议（工程化）

把 08 §5 的 V12（`reverse_calc_F002`）从"一个分布检验"细化为"**先分层、各组独立判、不汇总**"的协议：

```
V12 reverse_calc_F002（分层版）：
  1. 按 _subchannel 分组（main / extract / minor / dual_limit）
  2. 每组独立做：单 lot 倍率 + 跨 lot σ + 紧贴 LOQ 判定
  3. 每组用各自的阈值（main 的"紧贴"阈值 ≠ minor 的）
  4. 不跨组平均 / 不输出单一可信度分
  5. dual_limit 组走区间制判据（紧贴中点 / 跨 lot 一致），不套 F-002
```

**伪代码（确定性 · 无需 LLM）**：

```python
# 子通道边界 / 阈值：边界暂定（Q-035 押后 v0.5）；阈值取自量级，非全局常量
SUBCHANNEL = {
    "main_component":  lambda spec_min: spec_min >= 30.0,
    "extract":         lambda spec_min: 2.0 <= spec_min < 30.0,
    "minor_component": lambda spec_min: spec_min < 6.0,      # 有效成分
}
THRESHOLDS = {
    "main_component":  {"ratio_hi": 1.10, "sigma_floor_pp": 0.05},  # 弱信号 → 专门门槛
    "extract":         {"ratio_hi": 1.20, "sigma_floor_pp": 0.5},
    "minor_component": {"ratio_hi": 1.15, "sigma_floor_pp": 5.0},   # 离散是常态
}

def detect_f002_stratified(indicators, lot_groups):
    out = {}
    for sub, items in group_by_subchannel(indicators, SUBCHANNEL):
        thr    = THRESHOLDS[sub]
        ratios = [x.measured / x.spec_min for x in items]
        sigma  = cross_lot_sigma(items, lot_groups)         # 跨 lot 同物料同指标
        if sub == "main_component":
            # 主成分：紧贴 LOQ + 跨 lot σ≈0 = 反算嫌疑（弱信号，需专门低门槛）
            suspicious = mean(ratios) <= thr["ratio_hi"] and sigma < thr["sigma_floor_pp"]
        elif sub == "minor_component":
            # 微量：离散是常态；嫌疑来自"异常紧贴"或定性化退化（连倍率都算不出）
            suspicious = is_anomalously_tight(ratios, sigma) or is_qualitative_degenerate(items)
        else:
            suspicious = generic_extract_check(ratios, sigma, thr)
        out[sub] = Verdict(sub, suspicious, n=len(items),
                           mean_ratio=mean(ratios), sigma=sigma)
    return out      # 逐子通道结论 / 绝不汇总成单一分

def is_qualitative_degenerate(items):
    # 石膏"未过 X"无数值 / 汞跨物料统一填 LOD 边界值 = 无数可审（LOQ 边界退化）
    return any(x.value_is_qualitative or x.at_lod_boundary for x in items)
```

**工程契约**：每子通道的阈值（紧贴判定 / σ 下界）**取自该通道的量级结构**，而非全局常量——这与 08 的"σ 阈值取自访谈 ground truth"原则平行：08 的阈值来自领域知识，本文的阈值来自量级结构，两者都拒绝"一个常量套全部"。

**与 08 的接口**：08 §5 整合决策树的 C 分支（F-002 通道）只标"子通道倍率 + 跨 lot σ"；本文 §5 提供其**如何分层、各组用什么判据**的实现细则。10 是 08 的 V12 的"分辨率层"——08 给骨架，10 给分辨率。

## 6. 应用 — 中药提取案例 (case-2)

**lot-L1 / 7 物料 12 指标三子通道分布**（数据源 lot-L1-summary §3.1）：

| 子通道 | 指标（实例）| 倍率范围 | 单批判定 |
|--------|-----------|---------|---------|
| 主成分型 | 茯苓 β-葡聚糖 1.07x / 海螵蛸 CaCO₃ 1.043x | 1.04–1.07 | **suspicious**（贴线）|
| 浸出物型 | 5 个浸出物指标 | 1.25–1.68 | ambiguous |
| 微量型 | 甘草苷 1.93 / 芍药苷 2.83 / 延胡索乙素 1.975 等 | 1.33–2.83 | lean_actual |

**跨产品确证（P2 / F-003 condition B）**：main 最极端样本 = 石膏 CaSO₄·2H₂O **1.014x**（仅高 1.3pp）；minor 子通道跨产品分布**完全重叠**（P1均值 1.839 vs P2 1.833 / 差 0.006 / 范围重叠 78%）。"main 紧贴、minor 离散但同分布"——这两条跨产品同时成立，是 F-003 不是单产品偶然的确证。

**极差比矩阵**：55:1（单 lot）/ 136–370:1（跨 lot）——§4 稀释数学的实测值。

**定性化退化形态**（子通道的极端）：石膏重金属/砷只报"未过 X"（无数值可审，连倍率都算不出）；汞跨 4 物料（P1三七 + P2栀子/玄参/柴胡）统一报 0.1 mg/kg（50% LOD）——跨物种生理性富集差异应达 100 倍以上，统一填值是 LOQ 边界反算嫌疑。这是微量通道"难伪造"被推到极致后的塌缩：干脆不给数。

**判读结论**：主成分通道 suspicious、微量通道 lean_actual。**若不分层，这个"分裂结论"会被压成单一"ambiguous"，丢掉全部诊断力**——这正是本文方法相对朴素汇总审计的增量价值。

## 7. 跨域可迁移性

通用形式：**任何"阈值制 + 可填值 + 指标量级跨度大"的系统，操纵可检测性都随量级变化，检测必须分层；且最易操纵的通道往往最难抓。**

- **财报审计**：大科目（营收）的小百分比调整 vs 小科目（某项计提/减值）的调整——前者绝对额巨大但相对不显眼（易藏在正常波动里），后者相对差大却显眼。可检测性随科目量级变化；按量级分层比"对全表做一次 Benford 检验"更有分辨率。
- **学术 p-hacking**：紧贴 0.05 阈值的 p=0.049，在大样本（易微调到刚好显著）vs 小样本里的可信度不同；按样本量/效应量级分层，才看得清"贴线"是真效应还是凑出来的。
- **KPI / 绩效考核**：大基数指标的小幅注水难发现，小基数指标的注水显眼——考核审计若按"显眼度"抽查，会系统性放过大指标的系统性注水。
- **临床试验**：主要终点 vs 次要终点、大效应量 vs 小效应量的可操纵性与可检测性量级差。

**反转悖论的跨域版本**：在以上每个域，"最易被操纵的指标往往是最难抓的"。审计资源若按"异常显眼度"分配，会系统性投错地方——这是一条跨域的审计设计警告。

## 8. 与既有理论的关系（学术锚点 / 已 web 核实）

| 理论 | 与本文的接口 |
|------|------------|
| **信号检测理论 SDT**（Green & Swets 1966）| 可检测性 = 信噪比；不同子通道 = 不同 SNR 区。SDT 区分"敏感度(d′)"与"判定标准(criterion)"——本文反转悖论正是"审计的 criterion 按显眼度设，与真实 signal 所在错位"。|
| **Simpson 悖论 / 分层分析**（Simpson 1951；分层抽样传统 Cochran 1977）| 跨异质子通道汇总会抹掉甚至反转组内信号 = §4 全方差分解的核心。先分层再检验，是统计学对抗聚合失真的标准手段。|
| **法务数字/分布分析**（Benford 1938；Nigrini 2012）| 数值审计中"分布偏离 = 操纵信号"的先例。但 Benford 是**量级无关**的首位数律——正好反衬本文贡献：把量级**分层**引入分布审计，捕捉 Benford 类全局检验会平均掉的子通道信号。|
| **信号理论 / 伪造成本**（Spence 1973 job-market signaling；Zahavi 1975 handicap）| "难伪造的信号才可信"。倒用于审计：微量通道伪造成本高 → 其值更可信；主成分伪造成本低 → 最不可信。为"易操纵⊥易暴露"反转提供理论根。（Zahavi 在生物学内有争议，主依托 Spence。）|
| **检测限 LOD/LOQ**（IUPAC：LOD≈3×S/N、LOQ≈10×S/N）| 量级分辨率的物理底；定性化退化（"未过"/统一填 LOD 边界）= 数据被压到 LOQ 之下、无值可审的极端形态。|

（注：Goodhart 定律已由 08 使用于"机制/方向"理论；本文聚焦"检测/测量"理论，与 08 §7.3、09 §8 的理论分工一致，不重复。）

## 10. 局限性

- **子通道边界值未定**：main ≥30% / minor <6% 是 2 产品数据点，n 小（ADR-034 §3.4 风险 1）。`Q-035`（main 是否从 ≥30 改 ≥10 / 是否加 high_minor、high_extract 子通道）押后 v0.5（ADR-035）——本文边界判据是**暂定**，§5 的 `SUBCHANNEL` 阈值会随数据调整。
- **dual_limit 子通道样本极少**（颠茄/玉米淀粉/明胶/包材各少量），其判据未充分实证。
- **反转悖论的前提**：依赖审计者同时看"绝对差 + 相对差"。若只看相对差（如纯 Benford 式检验），悖论形态会变——这条边界条件待 v0.3 严格论证。
- **不覆盖 F-004**：downward 压线（收率/工艺参数）靠跨批 σ + 紧贴边界 + 访谈识别，**非**子通道分层。本文方法只作用于 F-002 的上限 upward 通道；把它套到 F-004 是方法越界。
- **推断边界（n=2 / 单企业数据）**：全部证据基于单企业·双产品，访谈为间接证据（非现场观察、非操作工本人）。F-004 X2 跨企业 documented_unresolvable；F-003 的跨企业稳健性同样未验证，generalizability 由 §7 跨域 + §8 理论间接补强，而非一手跨企业数据。对跨企业、跨品类的外推是**有方向的假说**而非已证命题。
- **识别 ≠ 证实（方法论自觉边界）**：子通道分层把"可检测性"提上台面，但**检测到 ≠ 证实**——贴线 + 低离散、定性化退化（连倍率都算不出）都只是"反算嫌疑"，证实须叠加访谈 / 现场观察 / 原始仪器数据。本方法定位为审计**第一道筛**，不替代制度性确证（与 08·09 同一自觉边界）。
- **阈值标定**：§4 的"何时必须分层"阈值、§5 的各通道阈值，目前取自 case-2 量级结构，跨域迁移时需重标。
