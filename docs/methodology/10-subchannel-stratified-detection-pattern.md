## 0. 论点与定位

**一句话论点**：在阈值制（≥ / ≤ / 区间）指标体系里，**操纵的可检测性是指标量级的函数**。统一阈值或汇总分析会系统性漏掉"最容易被操纵"的通道。检测必须按**量级同质的子通道**分层；不分层 = 把信号平均掉。

**本文的招牌贡献（signature contribution）**：「**易操纵 ⊥ 易暴露**」反转悖论 —— 越容易反算的通道（主成分型，紧贴 LOQ 看似合理），被操纵后信号越**弱**；越难反算的通道（微量型，微小绝对差却相对差极大），被操纵后信号越**强**。结果：朴素审计盯着"显眼"的微量通道（其实最可能真实），放过"不显眼"的主成分通道（其实最可能被操纵）——**检测注意力与操纵概率反向**。

**三篇分工（避免与 08/09 重叠）**：
- **09 = 结构**（合规叙事是什么 / 三层代差）
- **08 = 机制 + 方向**（反算如何发生 / upward F-002 + downward F-004 / 整合识别树）
- **10 = 检测分辨率**（如何才"看得见"反算 / 为什么必须分层 / 量级如何决定可检测性）

**四 finding 的 lens（案例护城河）**：
- **F-003**：本文主题。
- **F-002**：F-003 是 F-002 上限 upward 通道的量级精细化；本文的分层对象正是 F-002 通道。
- **F-001**：分层检测是 F-001"二阶检验"（判定表层是否反算）的分辨率工具。
- **F-004**：**边界**——F-004（downward 压线）靠跨批 σ + 紧贴边界 + 访谈识别，**不**靠子通道分层。本文明确不覆盖 F-004，避免越界。

## 1. 引子 — 为什么需要量级分层

**反例（不分层的失败）**：把所有"含量下限（≥）指标"当成一个桶，做统一的反算分布检验。主成分指标倍率 1.04–1.07x（紧贴 LOQ）与微量指标倍率 1.33–2.83x（远超 LOQ）被混在一起平均 → 整体看"分布正常、离散度健康" → **主成分通道的反算嫌疑被微量通道的天然离散稀释掉** → 漏判。

**实证锚点**：lot-L1 单 lot，主成分子通道极差 ~0.027 vs 微量子通道极差 ~9.98 → **极差比 ≈ 55:1**；跨 lot（L1→L2）扩大到 **136–370:1**。这是"汇总会抹掉信号"的直接度量。

**问题定性**：这是一个**检测分辨率**问题，不是检测方向问题（方向由 08 管）。同一种 upward 反算偏向，在不同量级通道呈现完全不同的可见度。

## 2. F-003 命题精确化

**核心关系**：反算难度 ∝ 1 / |标准值 × 允许误差|（即"绝对 headroom"越小，越难无痕反算）。

**子通道划分（当前 / 边界待定见 §10）**：

| 子通道 | 标准量级 | 实测倍率 | 反算难度 | 信号形态 |
|--------|---------|---------|---------|---------|
| 主成分型 main_component | ≥30%（实例 ≥50%/≥86%）| 1.01–1.07x | **低** | 紧贴 LOQ / 弱信号 |
| 浸出物型 extract | 2–30% | 1.25–1.68x | 中 | 中间形态 |
| 微量型 minor_component | <6%（有效成分）| 1.30–2.83x | **高** | 离散 / 强信号 |
| dual_limit（中间体/辅料/包材）| 区间制 | 紧贴中点 / 跨 lot 一致 | 低 | F-002 **不适用** |

**关键机理**：难度取决于"绝对差能否看似刚过线"。主成分 3–4 个百分点的差（53.5/50、89.7/86）看起来像合理过线；微量 0.04–2 个百分点的微小差，因相对差极大反而显眼——要么真测，要么大幅造假，没有"低调过线"的中间地带。

**与 08 的差异**：08 §3 用这张表**证明 F-002 反算存在且随量级变化**；本文用它**论证一条领域无关的检测原理**（量级决定可检测性），并把它发展为 §3 的反转悖论与 §5 的工程协议。

## 3. 检测悖论 — "易操纵 ⊥ 易暴露"（本文核心新贡献）

**反转**：操纵难度与操纵后的暴露度**反向**。
- 主成分型：**易操纵**（紧贴 LOQ 合理）→ 操纵后**难暴露**（信号弱）。
- 微量型：**难操纵**（微小绝对差难做得可信）→ 一旦操纵**易暴露**（信号强）。

**推论（反直觉）**：朴素审计按"哪里异常显眼"分配注意力 → 注意力流向微量通道（其实最可能是真测）→ 系统性放过主成分通道（其实最可能被操纵）。**检测注意力与操纵概率反向**——这是不分层检测的根本危险，比"信号被稀释"更深一层。

**为什么这是 stratification 必要性的深层理由**：不分层 = 用同一"显眼度"门槛扫所有通道 = 把注意力交给信号强度 = 被反转悖论系统性误导。分层 = 给每个量级通道一套**与该通道操纵难度匹配**的判据。

**理论连接（候选 / 待 v0.2 核实）**：costly signaling / handicap principle —— 微量成分像"高成本信号"（难伪造，故可信）；主成分像"低成本信号"（易伪造，故不可信）。**可检测性 = 信号伪造成本的函数**。

## 4. 信号稀释的数学 — 为什么"汇总"会抹掉信号

**直接度量**：极差比 55:1（单 lot）/ 136–370:1（跨 lot）= 汇总稀释的倍率。

**机制（Simpson's paradox / 分层分析的应用）**：跨异质量级子通道做汇总 σ / 分布检验时，主成分的"紧贴"（低 σ）被微量的"天然离散"（高 σ）平均掉 → 整体分布看似健康。子通道内：σ_main << σ_minor，但 pooled σ 介于两者、谁也不像。

**形式化方向（v0.2 展开）**：
- 定义 within-subchannel σ vs pooled σ；证明 pooled 检验对主成分反算的功效（power）随微量通道方差上升而下降。
- 给出"何时必须分层"的判据（子通道间量级跨度 / 方差比阈值）。

## 5. 整合 — 分层检测协议（工程化）

**把 08 §5 的 V12 重述为"子通道分层检验"**：

```
V12 reverse_calc_F002（分层版）：
  1. 按 _subchannel 分组（main / extract / minor / dual_limit）
  2. 每组独立做：单 lot 倍率 + 跨 lot σ + 紧贴 LOQ 判定
  3. 每组用各自的阈值（main 的"紧贴"阈值 ≠ minor 的）
  4. 不跨组平均 / 不汇总出单一可信度分
  5. dual_limit 组走区间制判据（紧贴中点 / 跨 lot 一致），不套 F-002
```

**伪代码骨架（v0.2 补全 / 确定性 · 无需 LLM）**：

```python
def detect_f002_stratified(indicators, lot_groups):
    out = {}
    for sub, items in group_by_subchannel(indicators):
        thr = SUBCHANNEL_THRESHOLDS[sub]          # 阈值按量级取，非全局常量
        ratios = [x.measured / x.spec_min for x in items]
        sigma  = cross_lot_sigma(items, lot_groups)
        out[sub] = judge(ratios, sigma, thr, near_loq=is_near_loq(ratios, thr))
    return out                                    # 逐子通道结论，不汇总
```

**工程契约**：阈值（紧贴判定 / σ 下界）**按子通道取自量级**，而非全局常量——与 08"阈值取自访谈"原则平行（08 阈值来自 ground truth；本文阈值来自量级结构）。

**与 08 的接口**：08 §5 决策树的 C 分支（F-002 通道）只说"子通道倍率 + 跨 lot σ"；本文 §5 提供其**如何分层**的实现细则。10 是 08 的 V12 的"分辨率层"。

## 6. 应用 — 中药提取案例 (case-2)

- **lot-L1 / 7 物料 12 指标三子通道分布**（数据源 lot-L1-summary §3.1）：主成分 2 个（茯苓 β-葡聚糖 1.07x / 海螵蛸 CaCO₃ 1.043x）= suspicious；浸出物 5 个 [1.25–1.68]；微量 5 个 [1.33–2.83]。
- **跨产品确证（P2 / condition B）**：main 最极端样本 = 石膏 CaSO₄·2H₂O **1.014x**（仅高 1.3pp）；minor 子通道跨产品分布**完全重叠**（P1均值 1.839 vs P2 1.833 / 差 0.006 / 范围重叠 78%）。main 紧贴、minor 离散但同分布 = F-003 跨产品确证。
- **极差比矩阵**：55:1（单 lot）/ 136–370:1（跨 lot）。
- **定性化退化形态**：石膏重金属/砷只报"未过 X"（无数值可审）；汞跨 4 物料统一报 0.1 mg/kg（50% LOD）——子通道的极端退化（LOD 边界反算 / 连倍率都无法计算）。
- **判读结论**：主成分通道 suspicious、微量通道 lean_actual —— 若不分层，这个"分裂结论"会被压成单一"ambiguous"，丢掉全部诊断力。

## 7. 跨域可迁移性

通用形式：**任何"阈值制 + 可填值 + 指标量级跨度大"的系统，操纵可检测性都随量级变化，检测必须分层。**

- **财报**：大科目（营收）的小百分比调整 vs 小科目（某项计提）的调整——前者绝对额大但相对不显眼（易藏），后者反之。可检测性随科目量级变化。
- **学术 p-hacking**：紧贴 0.05 的 p=0.049 在大样本（易微调）vs 小样本里"刚过线"的可信度不同；按样本量/效应量级分层才看得清。
- **KPI / 绩效考核**：大基数指标的小幅注水难发现，小基数指标的注水显眼。
- **临床试验**：主要终点 vs 次要终点的可操纵性与可检测性量级差。
- 反转悖论的跨域版本：**最易被操纵的指标，往往是最难抓的**——审计资源若按"显眼度"分配，会系统性投错地方。

## 10. 局限性

- **子通道边界值未定**：main ≥30% / minor <6% 是 2 产品数据点（n 小 / ADR-034 §3.4 风险 1）。`Q-035`（main 是否从 ≥30 改 ≥10 / 是否加 high_minor、high_extract）+ `Q-035` 边界子通道押后 v0.5（ADR-035）——本文边界判据是**暂定**。
- **dual_limit 子通道样本极少**（颠茄/玉米淀粉/明胶/包材各少量），其判据未充分实证。
- **反转悖论的前提**：依赖审计者同时看"绝对差 + 相对差"；若只看相对差，悖论形态会变（待 §3 严格论证）。
- **不覆盖 F-004**：downward 压线靠 σ + 紧贴边界 + 访谈，非子通道分层——明确边界，避免方法越界。
- **单企业数据**：F-004 X2 跨企业 documented_unresolvable；F-003 的跨企业稳健性同样未验证（generalizability 由跨域文献 §7/§8 间接补强）。
