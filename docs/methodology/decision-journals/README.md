# Decision Journals — D-route Q5 ACK 月度节律目录

> D-route §6.2 Q5 ACK 写作节律软约束：架构师每月 1 段决策日记 + 每季度 1 篇方法论文章。
> 本目录是月度日记部分（季度文章在 `docs/methodology/00-09*.md`）。

## 目录用途

按 D-route §6.2 + ADR-028 §2.3 Q5 ACK 节律约定：

- **每月 1 篇决策日记**（约 150-300 行 / 不超过 5000 字）
- **关注"元决策"** — 为什么这样决策、决策模式的反思、未来调整方向
- **不重复操作日志** — case-2 §10 进度日志 + sprint logs 是操作层 / 本目录是战略层
- **承认能力短板** — 按 CLAUDE.md §6.2，用户写作偏弱 / 架构师主动 propose 框架 + 起草初稿 / 用户角色为"审稿 + 决策 + ground truth 提供"

## 文件命名

`YYYY-MM.md` 格式 / 每月 1 个文件。

## 与既有文档的关系

| 文档类型 | 焦点 | 频率 |

| Methodology 季度文章（`00-09*.md`）| Pattern 抽象 + 方法论 | 每季度 1 篇 |
