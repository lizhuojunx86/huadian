## 2. 9 条独立年代区间建议

### 2.1 秦末

```yaml
- name: "秦末"
  start_year: -209
  end_year: -207
  midpoint: -208
  aliases: []
```

**学界依据**：
- 起点：秦二世元年七月（前 209）陈胜、吴广大泽乡起义，秦统治体系崩溃起始点（《史记·陈涉世家》、《资治通鉴·秦纪二》）

**Reference**：
- 《史记·秦楚之际月表》（中华书局 1959 标点本卷十六，序文明确"五年之间，号令三嬗"指此期）
- 《资治通鉴·秦纪二、秦纪三》
- 林剑鸣《秦汉史》（上海人民出版社 2003）第 2 编第 3 章

**与 PE draft 比较**：起讫一致 (-209 ~ -207, midpoint -208)，**无差异**。

### 2.3 战国·秦 / 战国秦 / 战国·韩 / 战国·魏（合并讨论）

四者均为"战国 + 国别"的 NER 标注变体，本质为战国时期的诸侯国君主/卿大夫细分。

#### 2.3.1 国别细分独立 period 路径（historian 倾向）

```yaml
- name: "战国·秦"
  start_year: -384  # 秦献公即位（栎阳定都为崛起起点）
  end_year: -221    # 秦统一六国（战国秦阶段终结）
  midpoint: -302
  aliases: ["战国秦"]   # 无中点符号变体合并

- name: "战国·韩"
  start_year: -403  # 三家分晋（韩立国）
  end_year: -230    # 秦灭韩
  midpoint: -316
  aliases: []

- name: "战国·魏"
  start_year: -403  # 三家分晋（魏立国）
  end_year: -225    # 秦灭魏
  midpoint: -314
  aliases: []
```

#### 2.3.2 战国 alias 化路径（PE draft 倾向）

PE draft 已将 4 项作为"战国"的 aliases（midpoint 一致 = -348）。

#### 2.3.3 路径选型

| 维度 | 独立 period | 战国 alias |
|------|-------------|-----------|
| midpoint 精度 | 高（战国·秦 -302 偏中后期 / 战国·韩 -316 / 战国·魏 -314 均贴各国实际活跃期） | 低（统一 -348，与战国·秦实际 midpoint 差 46yr，临近 50yr "alias 处理上限"） |
| 未来扩展性 | 高（如新增"战国·齐/楚/燕/赵"可遵循同 pattern） | 中（aliases 列表会越来越长） |
| R1 dynasty filter (200yr) 实际命中差异 | 微小（200yr 阈值下 -302 与 -348 差异不影响"是否拦截" 判定，仅影响 boundary case） | 同 |
| schema 维护 | 5 行新条目 | 4 行 alias 字符串 |

**Historian 建议**：
- **本 sprint 接受 PE alias 化路径**（PE draft 已经满足 R1 dynasty filter 当前需求）
- **登记本路径的局限**：当 R 规则未来需要更精细 dynasty distance（如 R2/R5 候选 100yr 阈值）时，"战国" 主区间 -348 与 "战国·秦" 实际 -302 的 46yr 差异可能进入"是否拦截"的 boundary 分歧区。届时再从 alias 提升为独立 period
- **NER variant 处理**：「战国秦」无中点符号，是 LLM v1-r5 输出的格式不一致变体。PE draft 已将其与「战国·秦」并列为 alias —— 同意。但建议未来在 NER prompt 层面 normalize（输入时统一为带中点的形式），避免 dynasty 字段值膨胀

**Reference**：
- 杨宽《战国史》（增订本，上海人民出版社 1998）卷一（年代框架）+ 卷四（战国诸侯纪年表）

### 2.4 春秋战国

```yaml
- name: "春秋战国"
  start_year: -481   # 《春秋》绝笔之年（获麟，鲁哀公十四年）
  end_year: -403     # 三家分晋
  midpoint: -442
  aliases: []
```

**学界依据**：
- "春秋战国" 作为标签通常指春秋末段至战国初段过渡期（约 80 年）
- 起点取 -481 派系：《春秋》绝笔之年，鲁哀公十四年获麟（《春秋·哀公十四年》"西狩获麟"为孔子绝笔，传统经学起点）
- 终点取 -403 派系：周威烈王二十三年，三家分晋（《资治通鉴》以此为战国开篇序，宋代以来主流断代）
- 区间 ~78 年，midpoint -442（约鲁悼公 / 越王勾践灭吴前后）

**alternative 起讫**（仅参考，不建议）：
- 《史记·六国年表》派：以前 475（周元王元年）为战国开端 → 春秋战国之交 = -481 ~ -475 太窄
- 部分史学派：以前 476（敬王崩、孔子卒）为春秋下限 → 区间 -476 ~ -403 = 73 年

**Reference**：
- 司马光《资治通鉴》卷一 周威烈王二十三年开篇序（"威烈王二十三年，初命晋大夫魏斯、赵籍、韩虔为诸侯"）
- 《史记·六国年表》序
- 童书业《春秋史》（中华书局 2005 修订本）"春秋下限说" 节
- 杨宽《战国史》卷一（"战国开端三说" 节）
- 钱穆《先秦诸子系年》（商务印书馆 2001）卷首"战国年代考辨"

**与 PE draft 比较**：

| 项 | PE draft | Historian 建议 | 差异 |
|----|---------|---------------|------|
| start_year | -500 | **-481** | **PE 早 19 年** |
| end_year | -450 | **-403** | **PE 早 47 年** |
| midpoint | -475 | **-442** | **PE 早 33 年** |
| 区间宽度 | 50 年 | 78 年 | PE 偏窄 |

**差异点 #3**（待架构师裁决）：

PE 注释自述取"折中区间 -500 ~ -450"，但学界主流断代（《资治通鉴》派 + 《史记·六国年表》派）均不在此区间内：
- PE -500 早于《春秋》绝笔（-481）19 年，亦早于通行春秋下限说
- PE -450 早于三家分晋（-403）47 年，落在"战国早段"内

**Historian 强烈建议修订为 -481 ~ -403**（学界标准两端点）：
- 起点 -481：经学传统（《春秋》绝笔）
- 终点 -403：宋代以来主流（《资治通鉴》三家分晋）
- midpoint -442：约鲁悼公元年 / 越灭吴前后，正处春秋战国之交

## 5. 复核方法学说明（供未来 historian-review 参考）

### 5.1 优先级排序

复核 dynasty 区间时，严格度排序：
1. **学界共识断代**（如秦统一前 221、平王东迁前 770、三家分晋前 403）→ 必须采用
2. **典型纪年事件**（如《春秋》绝笔前 481、王朝建立帝号、秦始皇崩前 210）→ 强力优先
3. **学派分歧断代**（如春秋下限三说、战国上限三说）→ 选**主流派 + 注明 alternative**
4. **过渡期模糊标注**（如"西周/春秋""春秋战国""战国末秦初"）→ 取分界事件前后 50-80 年覆盖区间

### 5.2 midpoint 取值哲学

- midpoint 用于 R 规则 dynasty distance 判定，不是历史"中位年代"语义（不是该时期最重要事件的年代）
- 因此严格按 yaml 文件头约定 `midpoint = floor((start_year + end_year) / 2)`
- 如 start_year + end_year 为奇数（如 -209 + -202 = -411），floor(-205.5) = -206

### 5.3 alias vs 独立 period 决策准则

- midpoint 差 < 50yr → alias 化（接受精度损失，简化映射表）
- midpoint 差 50-100yr → boundary case，按"未来规则严格度"决策
- midpoint 差 > 100yr → 必须独立 period

战国国别细分（秦/韩/魏）midpoint 差 32-46yr，处 alias 化区间，但临近上限 → 增补 future-risk 注释
