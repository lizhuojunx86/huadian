## 0. 为什么需要这份文档

按 case-2 评估文档 §3 的核心命题——**"规程外的经验才是 quality-decisive"**——要做实它，必须先精确划定**"规程内"的边界**：哪些参数被规程明文限定？限定到什么精度（点值 / 区间 / 仅描述）？哪些环节规程**没说**？

本文档将三个品种的提取工序规程**关键参数**抽取出来，每条都标注精度类型，作为后续比对每批批生产记录时的金标准。任何批记录中出现而本文档中不出现的参数，都是"规程外足迹"的候选。

| 浓缩终点 | "浓缩至适量" — **未量化** | 减压浓缩成稠膏 — **未量化稠度/相对密度** | **相对密度 1.35-1.40 (50℃)** — 量化区间 |
| 醇沉 | 无 | **含醇量 50%**（明文） | 无 |
| 干燥温度 | 未明文限定（成品干燥后粉碎） | **60℃ 以下干燥**（明文） | 未明文限定 |
| 制得稠膏/干膏 | 干膏粉 **800 ± 2% kg** | 稠膏 **360 ± 2% kg** | 稠膏 **522 ± 2% kg** |
| 投料比例（实际 / 规程批量） | 162/200 = **0.81** | 待 OCR 确认 | 待 OCR 确认 |

> **粗体**为规程明文给出的可量化数值或区间；非粗体为规程仅描述未量化或未限定。

## 2. P1（P1-CODE）提取规程内参数详表

### 2.1 处方（提取批 200 万粒）

| 物料 | 规程数量 (kg) | 实际批 162 万粒按 0.81 折算 | 实际批记录领料单数量 (kg) | 偏差 |
|------|---------------|--------------------------|--------------------------|------|
| 甘草 | 635.0 | 514.35 | 512.4 + 1.95 = **514.35** | **0%** |
| 白芍 | 635.0 | 514.35 | 510.45 + 3.90 = **514.35** | **0%** |
| 茯苓 | 476.2 | 385.72 | 383.19 + 2.53 = **385.72** | **0%** |
| 白及细粉 | 476.2 | 385.72 | 385.16 + 0.56 = **385.72** | **0%** |
| 三七细粉 | 19.8 | 16.04 | 15.67 + 0.37 = **16.04** | **0%** |
| 海螵蛸细粉 | 63.4 | 51.35 | 51.15 + 0.20 = **51.35** | **0%** |
| 延胡索净料 | 317.4 | 257.09 | **257.09** | **0%** |
| 颠茄浸膏 | 4.2 | 3.40 | 待 OCR 后续页 | — |

> **关键发现**：B1 批的领料单数量与规程按 0.81 比例折算的预期值**精确吻合到第二位小数**。这意味着投料数量这条线被严格执行——这是"规程内基线" anchor 的有效性首次验证。
> 然而：投料数量准 ≠ 提取过程参数准。后续 OCR 重点应在工艺参数页（料液比、提取时长、温度曲线、浓缩终点、设备实际参数）上看实际值是否同样紧贴规程。

### 2.2 提取工艺规程内参数（按规程 §4.3 制法）

- **粉碎类**：白及、三七、海螵蛸 → 细粉（细度规程未在制法段限定，需查岗位 SOP `SOP-A/GW/02101` 前处理粉碎规程）
- **甘草水煎**：煎煮 **4 次**，第 1-2 次各 **3 小时**，第 3-4 次各 **2 小时**；料液比、温度未在主规程量化
- **白芍/延胡索/茯苓水煎**：加 **6 倍量水**，煎煮 **2 次**，第 1 次 **3 小时**，第 2 次 **2 小时**
- **滤液处理**：合并、板框滤过、浓缩"至适量"
- **加入物料**：加入细粉 + 颠茄浸膏，搅匀
- **干燥+粉碎**：粉碎成细粉

### 2.3 规程明文未限定但实际操作必然涉及的"暗参数"（candidate invariants 候选）

> 这些是后续 OCR 时需要重点抓取的字段，每一项都是 case 命题的潜在证据：

- 甘草水煎的**料液比** — 规程仅给白芍组的"6 倍量"，甘草组未明文
- 提取**温度**（沸 / 微沸 / 持续沸腾） — 规程默认水煎温度
- **浓缩终点**的判定方式（相对密度？目测稠度？体积？）
- **滤过参数**（板框型号、滤布目数）
- **设备型号**实际使用了哪台
- 是否存在**浸泡前置步骤**（操作工经验类操作）
- 投料**顺序与方式**（是否分批投入、投入速率）
- 第一次煎煮**升温速率**

## 3. P3（P3-CODE）提取规程内参数详表

### 3.1 工艺特殊性

P3是三品种中**最复杂的**——22 味药材加 2 个辅料，工艺涉及四条独立分支：

1. **超临界 CO2 萃取**（卵磷脂制备）— 蛋黄颗粒 → 超临界萃取罐 → 乙醇夹带剂提取 → 回收乙醇 → 70℃ 以下减压干燥 → 卵磷脂稠膏
2. **醇提回流**（12 味药材：丹参、党参、黄芪、石菖蒲、制何首乌、熟地黄、白芍、枸杞子、香附、远志、酸枣仁、黄芩）— **75% 乙醇浸泡 16 小时** → **加热回流 3 次** → 过滤 → 回收乙醇 + 减压浓缩成稠膏
3. **水提 + 醇沉**（7 味：山药、茯神、麦冬、葛根、粉葛、龙骨、菊花）— 水煎 2 次 → 滤过 → 浓缩至适量 → 加乙醇至**含醇量 50%** → 放置 → 滤过 → 减压浓缩成稠膏
4. **细粉直投**（人参 → 100 目细粉）

### 3.3 规程明文已量化的参数

- **乙醇浓度 75%**（醇提）/ **95%**（卵磷脂萃取夹带剂、维生素E溶解）
- **浸泡 16 小时**
- **超临界 CO2 萃取**：温度 42-50℃，压力 30 MPa，CO2 流量 15-20 L/h，时间约 3 小时
- **超临界乙醇夹带剂**：12-15 倍量 95% 乙醇
- **醇沉含醇量 50%**
- **干燥 70℃ 以下**（卵磷脂）/ **60℃ 以下**（终干燥）
- **粉碎细度**：人参细粉 100 目过 80 目筛；其他粗粉 10 目筛网
- **干燥温度**：药材净选后 80-85℃（多数）/ 山药 85-90℃ / 干燥时间 200kg 4-5 小时

### 3.4 规程未量化的"暗参数"（候选）

- 乙醇回流**单次时长**（仅"加热回流三次"，未给每次时长）
- **回流温度**（未明文，默认乙醇沸点附近）
- 各醇提组**料液比**（24000 kg 95% 乙醇对应总粗粉量约 1959 kg，倍量约 12.25，但未在规程明文限定）
- **浓缩相对密度终点**（仅"减压浓缩成稠膏"）
- **超临界萃取乙醇夹带剂的注入速率**
- 维生素E**喷入方式**（喷雾器型号、喷入压力、喷入时干燥状态）

## 4. P2（P2-CODE）提取规程内参数详表

### 4.1 双设备可选 — 重要的"规程内"分歧点

P2规程是三品种中**唯一明文给出多套设备方案**的：

| 方案 | 设备 | 投料分组 | 单罐投料量 | 料液比 | 提取条件 |
|------|------|---------|-----------|-------|---------|
| A | **E5 多功能提取罐** | 4 等份 | 341 kg/罐 | **8 倍量水** | 提取 **2 次 × 1.5 h** |
| B | **E6 回流提取装置** | 2 等份 | 682 kg/罐 | **8 倍量水** | 提取 **2 次 × 1.5 h** |

**关键 invariant 候选**：批记录里**实际用了哪台**？同一批量的 6 批之间是否设备稳定，还是混用？设备选型与提取得率/含量是否相关？这是用户提到的"提取设备种类形态影响"在规程层面的真实显现。

| 物料 | 规程数量 (kg) | 备注 |
|------|--------------|------|
| 玄参 | 222.28 | 入水提 |
| 龙胆 | 147.74 | 入水提 |
| 石膏 | 444.57 | 入水提 |
| 柴胡 | 147.74 | 入水提 |
| 栀子粗粉 | 147.74 | 入水提 |
| 知母 | 147.74 | 入水提 |
| 黄芩 | 106.42 | 入水提（**仅占总黄芩量的 35%**） |
| 黄芩细粉 | 197.28 | **65% 黄芩另制成细粉，制剂阶段加入**（不入提取） |
| 薄荷脑细粉 | 2.88 | 制剂阶段加入 |
| 硬脂酸镁 | 5.00 | 制剂阶段加入 |
| 制得稠膏 | 522 ± 2% kg | — |

### 4.3 规程已量化的提取参数

- **加水量**：8 倍量
- **提取次数**：2 次
- **每次时长**：1.5 小时
- **浓缩终点**：相对密度 **1.35-1.40 (50℃)** ✅ 这是三品种中**唯一**给出量化稠度终点的
- 药材切制规格：玄参 1-2mm 片 / 龙胆 5-10mm 段 / 柴胡 2-4mm 片 / 栀子 / 知母 / **黄芩 1-2mm 片**（往复式切药机）
- 干燥温度：60-80℃（绝大多数药材）
- 黄芩细粉过 100 目筛
- 薄荷脑过 80 目筛

### 4.4 暗参数

- 实际选用的提取设备型号（A vs B）
- 升温至沸的速率
- 沸腾保持的判定方式
- 滤液合并与浓缩之间的转移时间
- 黄芩拆分 35% / 65% 的实际比例是否精确（**重要**，因为黄芩素含量是P2的主要疗效指标）

## 5. 三品种共有的"规程外"领域（普遍候选 invariants）

按 5M 框架交叉，下面这些是**三品种规程都没说但实际操作必然存在**的暗领域：

### 人 (Person)

- 提取罐启动前的**预热/预清洁**步骤是否一致
- 投料**顺序**（先粉碎物料 vs 先液态溶剂）
- 沸腾判定的**主观尺度**（哪一种"翻腾状态"算到位）
- 对**异常**的现场处置（颜色异常 / 气味异常时是否记录、如何决策）

### 机 (Machine)

- 设备**校准状态**（温度计、压力表、流量计、PH 计的最近校准时间）
- 设备**结垢/残留**（前批残留对当前批的潜在污染）
- **加热升温曲线**的实际形态（蒸汽阀门开度、夹套温度 vs 物料温度差）

### 料 (Material)

- 药材**实际含水率**（影响加水倍量的实际溶剂浓度）
- 药材**颗粒粒度分布**（规程仅规定切制毫米数，分布未规定）
- 同一规程批号下**不同小批的均匀性**

### 法 (Method)

- 规程未量化的所有"至适量"、"适量"
- **过滤压力**与**过滤速率**
- **滤布**的目数与更换频次
- **泵送速率**与**管路温降**

### 环 (Environment)

- 提取工序车间**温度与湿度**实际值
- **季节**（影响药材初始含水、影响溶剂蒸发损失）
- **生产时段**（白班/夜班的设备稳定性差异）

## 6. 接下来 OCR 阶段的参数捕获优先级
