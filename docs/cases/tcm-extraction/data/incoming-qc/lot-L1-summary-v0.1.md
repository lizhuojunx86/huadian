### 2.1 关键观察

**(A) 杂质/药屑全部 33.3% LOD ⚠️**
7 物料杂质实测全部为 1%，标准全部为 ≤3%。这种"完美一致"在物理上不太可能——不同药材形态、产地、加工过程千差万别，杂质率应有显著差异。
- 解读 1（compliance）：杂质项是肉眼粗判，操作工按"本批看起来还行"统一填 1%，是 F-001 表层
- 解读 2（actual_low_resolution）：检验仪精度只到 0.5% 或 1%，所有 ≤1% 都报 1%，是真实但分辨率不足
- 待 6 批数据看是否仍稳定 1%

**(B) 汞水平跨物料差 100 倍（A-004 / 强 actual）**
延胡索 0.001 mg/kg → 三七 0.1 mg/kg = 100x。
三七是已知易富集汞的根类药材；这种跨物种生理性差异不可能被反算合成。

**(C) 海螵蛸镉/砷限值放宽 5 倍 → 实测仍极低**
镉限值 5（vs 饮片 1）实测 0.02 → 0.4% LOD；砷限值 10（vs 饮片 2）实测 0.05 → 0.5% LOD。即使限值大幅放宽，海螵蛸实测仍远低于饮片实测的绝对值——动物源材料本就低重金属。

**(D) 跨物料分散度判定**
- 三七 9 维分布 [4, 5, 6, 20, 33.3, 50, 55, 57.8, 73.3] → 极差 69.3 个百分点 → 强 actual
- 延胡索 8 维 [0.5, 2, 5, 5, 20, 33.3, 62, 62.5] → 极差 62 → 强 actual
- 白芍 8 维 [4, 5, 12, 15, 25, 33.3, 60, 60] → 极差 56 → actual
- 甘草 8 维 [3, 5, 10, 20, 20, 33.3, 44, 67.5] → 极差 64.5 → actual
- 海螵蛸 6 维 [0.4, 0.5, 15, 20, 25, 33.3] → 极差 32.9 → actual
- 茯苓 3 维 [33.3, 60.7, 70] / 白及 3 维 [33.3, 62, 64] → 维度少分散弱

**单批整体上限通道判定**：6/7 物料 → actual（茯苓/白及因维度少 → actual_low_n）

### 3.1 形态分类（A-003 关键发现）

按下限指标的"标准量级"分子通道：

| 子通道 | 标准量级 | 所含指标 | _倍率 范围 | 单批 layer |
|--------|---------|---------|-----------|-----------|
| **主成分型** | ≥50% / ≥86% | β-葡聚糖, CaCO3 | **1.04-1.07** | suspicious_main_component |
| **浸出物型** | 2.5-22% | 5 个浸出物指标 | 1.25-1.68 | ambiguous |
| **微量成分型** | 0.04-6% | 5 个有效成分 | 1.33-2.83 | ambiguous_lean_actual |

**核心解读**（A-003）：
- 主成分型反算容易：3-4 个百分点的"刚过线"看起来合理（53.5/50, 89.7/86）
- 微量成分型反算困难：绝对差 0.04-2 个百分点，相对差极大反而显眼；要么真测要么大幅造假
- 这两种形态在同一份 lot-L1 数据里同时出现，说明**反算者（如果存在）的能力或动机在两种指标上不同**

### 3.2 v0.3 → v0.4 子通道升级候选

A-003 的实证基础足够提议：F-001 二阶检验下限通道应进一步分子通道为"主成分 / 浸出物 / 微量"三型，每型的 6 批分布判定函数不同。

**触发 v0.4 升级条件**：6 批同物料分布检验时，主成分组的 σ 显著小于微量成分组 σ（如 σ_主 < 0.5% 且 σ_微 > 5%）→ A-003 升格为 F-003 fundamental finding 候选。

**当前状态**：A-003 = `tentative_finding_promote_candidate`，**等用户批准 v0.4 schema 升级前不动 schema-v0.1 主文档**。

## 4. lot 闭环 + 加工损耗发现

### 4.1 lot 链路二段结构

7 物料按上下游 lot 模式分两组：

**A 类（饮片直投，链路一段）**：
- 甘草饮片 L1 → 直接投料（领料单 lot 仍是 L1）
- 白芍饮片 L1 → 直接投料
- 茯苓饮片 L1 → 直接投料

**B 类（饮片→中间体细粉/净料，链路二段）**：
- 白及饮片 L1 → 内部加工 → 白及细粉 Q20260101 → 投料
- 三七 L1 → 三七细粉 Q20260101 → 投料
- 海螵蛸 L1 → 海螵蛸细粉 Q20260101 → 投料
- 延胡索 L1 → 延胡索净料 Q20260101 → 投料

**lot 命名规则**：`Q-prefix` 前缀 = 内部中间体 lot；纯数字进厂批号 = 外购 lot。

### 4.2 B 类加工损耗一致性（新发现）

| 物料 | 来料 kg | 投料 kg | 名义投料率 | 加工损耗率 |
|------|---------|---------|-----------|-----------|
| 三七 | 100 | 16.04 | 16.0% | 84% |
| 海螵蛸 | 337 | 51.35 | 15.2% | 84.8% |
| 延胡索 | 1693 | 257.09 | 15.2% | 84.8% |
| 白及 | 2431 | 385.16 | 15.8% | 84.2% |

**惊人一致**：4 个 B 类物料投料率全部在 15.2-16.0% 之间（极差 0.8 个百分点）。

**两种解读**：
1. **真实工艺特征**：粗粉碎 + 多次过筛 + 杂质去除 + 含量纯化的标准流程在 4 种不同药材上确实有相近损耗——但生理结构差异很大的根（三七/延胡索） vs 块茎（白及） vs 动物骨（海螵蛸）有相同 84% 损耗，物理上不太可能
2. **反向计算填表**：投料量 = 来料量 × 0.16 是个简化公式，反算从一个目标投料量除回去得来料量；这条解释更可能

⚠️ **若 #2 成立**：来料量本身可能也是反算的，这会**严重削弱**来料检验作为"中层 ground truth"的地位 → F-001 战场需要扩大到"来料数量"字段本身。

但需要谨慎：A 类（饮片直投）使用率也都是 16.7%，可能是企业生产计划的"6 批均摊"惯性而非反算。

**Open Question Q-019**（新增候选）：B 类加工损耗 84-85% 是真实工艺常数还是反算 artifact？需要现场看实际加工车间记录、测重日志，或问用户。

## 6. 单批 7 物料整体 layer 综合判定（v0.3 双通道）

```
batch_lot_L1_combined_layer = combine(
    上限通道: 6/7 actual + 1 actual_low_n_可观察 → 整体 actual,
    下限通道: 主成分子通道 2 个 suspicious + 浸出物/微量子通道 10 个 ambiguous → 整体 mixed
)
= mixed (上限 actual / 下限 mixed)
```
