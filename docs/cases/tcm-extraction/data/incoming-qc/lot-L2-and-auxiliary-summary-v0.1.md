### 2.3 极差比：main 紧贴 vs minor 波动

```
main_component 子通道跨 lot 相对差范围: 0% → 0.22%
minor_component 子通道跨 lot 相对差范围: 8% → 30%
extract 子通道跨 lot 相对差范围: 0-3% → 5-15% (中间形态)

极差比 = minor 上限 (30%) / main 上限 (0.22%) ≈ 136:1
极差比 = minor 上限 (30%) / main 中位 (0%, e.g. 黏度 / 干燥失重) ≈ ∞
```

⚠️ **F-003 候选**：从 lot-L1 单 lot 横向极差比 55:1 升级到**跨 lot 双向实证**极差比 136:1 ~ ∞:1。形态完全符合 F-003 命题：main 反算容易 → 跨 lot 紧贴；minor 反算困难 → 跨 lot 真实波动。

**F-002 v0.3 候选限定**：
- F-002 命题（"含量标准只设下限 → upward 反算偏向"）适用于**中药材成品/饮片含量测定**
- 不适用于中间体 / 辅料 / 包材 — 这些类有 dual_limit 标准，"高即好"话语不存在

**dual_limit 反算嫌疑新形态**：
- 紧贴上限 → "actual_high"
- 紧贴下限 → "suspicious_LOQ"
- **紧贴中点 → "安全居中反算"**（颠茄浸膏 9.7 vs 中点 9.65 偏离 0.05 / 玉米淀粉 pH 5.8 vs 中点 5.75）
