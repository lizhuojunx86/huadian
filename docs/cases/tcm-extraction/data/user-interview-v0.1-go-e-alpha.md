## 0. TL;DR

**核心发现**：F-003 命题在批生产记录侧的反算机制**与 F-002 来料检验侧不同**：
- F-002（来料检验）：文化话语 upward 偏向（"含量高=质量好"）
- F-003 修订（批生产记录）：**操作工博弈论 downward 压线**（"留未来失败空间"）

这是一个**比 F-002 更深的发现** — 反算不仅有文化驱动，还有博弈论驱动。**F-004 候选浮现**："操作工压线合格机制"。

F-001 三层结构得到精细化：表层（合规叙事）分**输入参数反算**（投料/温度/时长）+ **输出参数工艺控制**（密度/含量）+ **衍生指标压线**（收率）三类机制。

## 1. Q-027（F-003 finalize 条件 C 关键）— 用户答案

### 1.1 用户原话

> "Q-027 我不是十分确定，但我知道提取浓缩环节不可能控制那么精确，而且 40% 并非终点，操作工往往将需检验的参数（例如相对密度）控制到位后，即便收率更高也不会写真实的数据，因为怕以后公司按照较高的收率考核他，也可能会被 QA 调查为什么收率那么高而以前很低。中成药批间差异常常很大，来料的差异可能相差更大，现场操作工的操作也可能很不严谨（中间刷手机，出去吸烟，甚至回家），导致收率下降。所以综合考量，一线操作工倾向于每次都压线合格，以便给自己留足未来的失败空间。"

### 1.2 用户揭示的"操作工压线合格"机制

**核心命题**：操作工有意识地选择写入"压线合格"的数据，原因：
- **怕被高基线考核**：若本批写真实较高收率 → 公司未来按此考核 → 自己未来失败时无安全垫
- **怕 QA 调查异常**："为什么这批高而以前低" → 引起额外审查
- **承认现实噪声大**：批间差异 + 来料变化 + 操作不严谨（刷手机/吸烟/回家）→ 真实收率波动 σ 应当 > 0.5%
- **理性策略**：每批都压在规程下限附近 → 给自己"未来失败空间"

### 1.3 F-003 finalize 影响

**条件 C 评估**：
- 用户答案**确认**操作工有意识地压线 ≠ 真实工艺极稳
- 但反算的**机制**不是 F-002 命题（文化话语 upward）— 而是**新机制**（博弈论 downward 压线）

**F-003 命题状态**：
- 原命题：F-002 反算难度随指标量级不同（来料检验下限指标 1.04-2.83x 分子通道分裂）→ ✅ 仍成立（lot-L1+L2 数据支持）
- 修订条件 A：批生产记录工艺参数 σ ≈ 0 + 紧贴下限 = 反算嫌疑 → ✅ 用户 Q-027 答案直接确认
- 但**反算机制**：F-003 主要是 F-002 的延伸（文化话语扩展到工艺参数）？还是新机制（博弈论压线）？

**最终判定**：F-003 命题在批生产记录侧的实证机制**与 F-002 不同**。原 F-003 是 F-002 的子通道细化，但 Q-027 揭示批生产记录侧的反算是**独立于 F-002 的新机制** → 新 finding F-004 候选浮现。

**F-003 升级**：`candidate_condition_A_revised_met_pending_B_C` → **`finalized_in_来料检验侧_with_mechanism_extension_F004`**

### 1.4 F-004 候选 fundamental finding

**F-004 候选 claim**："**操作工压线合格机制**" — 中药批生产记录中的反算 downward 压线（不仅是 F-002 的文化话语 upward）。操作工出于自我保护策略（怕考核、怕调查）有意识地把可调整指标写入规程下限附近，给自己未来失败时留出"安全垫"。该机制与 F-002 的文化话语驱动并存，且**适用方向相反**：

| | F-002 | F-004（候选）|
|--|-------|------------|
| 反算方向 | upward（推高）| **downward（压低 / 压线）** |
| 适用 section | 含量测定下限指标 | 收率 / 工艺参数 / 衍生指标 |
| 驱动 | 文化话语（"高即好"）| **博弈论自我保护**（怕考核 + 怕调查）|
| 触发 | 心智偏向（无意识）| **理性策略**（有意识）|
| 数据形态 | 倍率 1.04-2.83x | σ ≈ 0 + 紧贴下限 |

**升级到 finalized 的条件**：
- 跨产品扩展确认（参乌 / P2的收率是否同样紧贴下限？）
- 跨企业访谈确认（其他中成药企业的操作工是否有同样行为？）

**broader value**：F-004 是一个**博弈论 + 行业心理学**的新方法论命题 — 在合规叙事审计中，需要识别**操作工 / 一线人员的自我保护策略**对数据偏向的影响。huadian D-route Layer 2 方法论文章潜在选题之一。

## 4. Q-028 — 用户答案: 输入参数倒推 / 输出参数不造假

### 4.1 用户原话

> "Q-028 我不确定，但实践中输入参数往往是倒推的'合理'数值（例如投料数量，温度，时长等），输出参数一般不会造假（例如相对密度），因为 QC 要独立检测并出具报告，这个产物还需要下一个环节（制剂）来使用，如果造假了下一个环节会有意见。"

### 4.2 用户揭示的 F-001 表层精细分层

**关键区分**：
- **输入参数**（操作工自填，无外部独立验证）→ **倒推合理数值**填表
  - 投料数量、温度、时长、料液比、打药液时间
- **输出参数**（有 QC 独立检测 / 下游使用约束）→ **不造假**记录值
  - 相对密度（QC 复核）、含量测定（QC 检测）、外观（下一环节肉眼可见）
- **衍生指标**（介于输入和输出之间）→ 通过**工艺控制**让结果落到目标区间
  - 收率（= 浸膏重量 / 投料量）→ 操作工**控制浓缩程度**让收率压线，而非**修改记录**

### 4.3 与 Q-027 答案的整合

Q-027 + Q-028 = **完整的"操作工合规努力"模型**：

```
F-001 表层（合规叙事）拆解：
├── 输入参数（控制性记录）→ 倒推填表（F-002 / F-004 反算）
│   ├── 投料量 514.35 kg：反算到模板批量
│   ├── 提取温度 100°C：反算到规程标准值
│   └── 提取时间 3+2 h：反算到规程标准值
├── 输出参数（测量性记录）→ 真实记录（受 QC + 下游约束）
│   ├── 相对密度：QC 独立复核 → 不造假
│   └── 含量测定：QC 检测 → 不造假
└── 衍生指标（控制性结果）→ 工艺控制让结果落到目标区间（F-004）
    └── 收率 40.26-40.32%：操作工控制浓缩程度让浸膏量 ≈ 目标
```

⚠️ **F-001 三层结构 + Q-027/28 精细化** → 案例理论体系大幅丰富。

### 4.4 对相对密度两组模板（A-014）的解释

按 Q-028 用户答："输出参数一般不会造假"，那为什么相对密度跨 6 批两组各 3 批完全一致？

**可能解释**：
- 仪器精度天然到 0.01（相对密度计精度）
- 浓缩终点判定标准化（操作工有经验把终点控制到精确密度区间）
- 4 个测量时间点（17:20/40/50/18:00）反映规程要求的"末段每 10 分钟测一次"
- 两组数据反映**两个班次/工艺员组合**的密度控制偏好（一组偏 1.28 起点 / 另一组偏 1.26 起点）

**结论**：A-014 不是"模板复用反算"，而是**两组班次的工艺控制偏好**（与 Q-026 + Q-006 一致）。A-014 anomaly 应当**降级**：从"反算细粒度证据"修订为"班次工艺控制偏好"。

## 6. F-001 / F-002 / F-003 / F-004 命题体系（GO-E-α 后）

### 6.1 F-001（v0.4 精细化）

**v0.2 命题**：中药批生产记录是分层制品（表层/中层/深层）

**v0.4 精细化**（基于 Q-026 + Q-027 + Q-028）：
- **表层（合规叙事）三类参数**：
  - 输入参数（反算填表 / F-002 + F-004 驱动）
  - 输出参数（真实记录 / QC + 下游约束）
  - 衍生指标（工艺控制压线 / F-004 驱动）
- **中层（实际执行）**：车间主任 + 排产决策（运筹层 / 非个人灵活性）
- **深层（身体化经验）**：操作工现场决策（含工艺技巧 + 自我保护策略 + 真实不严谨 + 班次配合）

### 6.2 F-002（v0.3 保持 / 适用范围已限定）

**v0.3 命题**：含量标准只设下限 → upward 反算偏向（仅适用于中药材成品/饮片含量测定）

**GO-E-α 影响**：无变化。F-002 在来料检验侧的命题不变（v0.3 限定保留）。

### 6.3 F-003（finalize 在来料检验侧 / 命题机制扩展到 F-004）

**status 升级**：
- `candidate_condition_A_revised_met_pending_B_C`
- → **`finalized_in_来料检验侧`**（来料检验下限指标的子通道分裂得到 lot-L1+L2 双向实证 + 用户访谈 Q-027 间接确认）
- 修订条件 A（批生产记录工艺参数 σ ≈ 0）实证的反算机制**不归 F-003**，而是新 F-004 命题

**F-003 finalized 适用范围**（finalized 后的明确边界）：
- 适用：来料检验下限指标（含量测定）的 6 子通道（main / extract / minor + 3 类 dual_limit）
- 反算难度随指标量级变化（lot-L1 极差比 55:1 / 跨 lot 极差比 370:1）
- F-002 文化话语驱动是机制核心

### 6.4 F-004 新候选 fundamental finding（GO-E-α 浮现）

**F-004 候选 claim**："操作工压线合格机制" — 中药批生产记录中存在与 F-002 不同的反算机制：操作工出于博弈论自我保护策略（怕考核 + 怕调查），有意识地把**可调整指标**写入规程下限附近，给自己未来失败时留出"安全垫"。

**与 F-002 的区别**：
- F-002：文化话语 upward 反算（"高即好"，无意识）
- F-004：博弈论 downward 压线（"压线合格 + 留安全垫"，有意识）

**适用范围**：
- 批生产记录的"衍生指标"（收率 / 浸膏重量等）
- 可能扩展到其他需要"控制到规程区间"的工艺参数

**实证基础**（GO-D-α + GO-E-α）：
- 6 批P1收率 σ = 0.022 pp + 全部紧贴 40% 下限
- 用户访谈 Q-027 直接确认机制（"操作工倾向于每次都压线合格"）
- 用户揭示真实生产噪声大（来料差异 + 操作不严谨）→ σ 应当 > 0.5% → 强反算确证

**finalize 条件**：
- 条件 X1：跨产品复现（参乌 / P2的收率是否同样紧贴下限）⏳ 未做（Q-017）
- 条件 X2：跨企业实证（其他中成药企业的操作工是否有同样行为）⏳ 未做

**broader methodological value**：
- 跨案例可迁移：任何"有外部考核压力 + 一线人员有自填权限"的领域都可能出现
- 行业例：财报压线达标 / 学术 p-hacking 但避免过度显著 / 临床试验主要终点压线 P=0.04999 等
- 命题形式化：**"反算难度 + 操作者自我保护策略 + 外部考核机制"三者交互**
- huadian D-route Layer 2 方法论文章**核心候选选题**：
  - "AI 知识工程在合规叙事审计中的双重反算识别框架（F-002 文化话语 + F-004 博弈论）"
