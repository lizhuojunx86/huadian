# docs/audits/

Architecture audit reports for the HuaDian project.

Each report is a point-in-time snapshot covering compliance checks
against the project constitution (`docs/00_项目宪法.md`), ADR adherence,
and code-level risk assessment.

## Naming convention

`audit-YYYY-MM-DD.md` — one report per audit session, dated by execution day.

## Scope

- Constitution compliance (C-1 ~ C-21)
- ADR implementation status
- Code quality / risk flags (PASS / WARN / FAIL)
- Read-only: audits never modify source files
